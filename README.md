# Shopez - E-commerce Web Application

Shopez is a modern, responsive e-commerce web application built with **React** for the frontend and designed to provide seamless shopping experience. It allows users to browse products, search, add to cart, checkout, and track orders. Admin users can manage products and orders.

## Features

### User Features
- User registration and login  
- Search products by name  
- Add products to cart  
- Checkout with address details  
- View order history  

### Admin Features
- Add new products  
- Edit existing products  
- Manage and update order status
